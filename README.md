# My Next.js Starter

>⚡️ 一个 Next.js (v15) 快速启动模板, 集成 Tailwind CSS (v4) + React (v19) + Auth.js + Prisma + TypeScript

## 🧑‍💻 安装和运行

- 安装依赖

```bash
pnpm i
```

- 本地开发

```bash
pnpm dev
```

接着用浏览器打开 http://localhost:3000 即可访问服务
